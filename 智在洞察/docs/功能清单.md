# "智在洞察"功能清单

**版本**: v1.0  
**更新日期**: 2026年3月17日  
**排序原则**: 按P0-P2优先级排列，同级别按依赖关系排序

---

## 优先级说明

| 级别 | 含义 | 说明 |
|------|------|------|

## 一、P0 核心功能（MVP必备）

### 1.1 数据采集模块

#### F-001 专项债信息爬取
- **功能描述**: 自动爬取中国债券信息网、地方政府债券信息公开平台的专项债信息
- **采集内容**:
  - 债券发行公告（规模、期限、利率）
  - 项目清单（项目名称、投资额、建设内容）
  - 实施机构信息
  - 资金使用计划
- **采集频率**: 每日增量更新
- **技术要求**: 支持多数据源、增量爬取、异常告警

#### F-002 政策文件爬取
- **功能描述**: 爬取国家发改委、财政部、工信部等部委及地方政府政策文件
- **采集内容**:
  - 政策文件原文
  - 政策解读文章
  - 产业规划文件
- **关键词过滤**: 专项债、两新、新型基础设施、智慧城市、数字化转型等

#### F-003 招标公告爬取
- **功能描述**: 爬取各大招标平台与运营商相关的招标信息
- **数据源**: 中国采购与招标网、各省公共资源交易中心、运营商电子采购平台
- **过滤维度**: 
  - 行业：通信、信息化、智慧城市
  - 金额：500万以上项目
  - 地区：支持多省份配置

### 1.2 数据处理模块

#### F-004 信息结构化解析
- **功能描述**: 将非结构化文档（PDF、Word）解析为结构化数据
- **解析维度**:
  - 项目基本信息（名称、金额、地点、周期）
  - 建设内容关键词提取
  - 技术需求分析
  - 潜在合作方识别
- **技术要求**: 支持OCR、NLP关键词提取

#### F-005 项目分类标签
- **功能描述**: 对采集的项目进行自动分类打标
- **标签体系**:
  - 一级分类：新基建、智慧城市、行业数字化、公共服务
  - 二级分类：5G、数据中心、智慧交通、智慧医疗等
  - 项目阶段：规划、立项、招标、实施
  - 商机状态：潜在、跟踪、投标、中标

### 1.3 商机分析模块

#### F-006 商机智能匹配
- **功能描述**: 根据项目信息与运营商业务匹配度识别商机
- **匹配规则**:
  - 项目类型与业务能力匹配
  - 地区与分公司覆盖匹配
  - 时间与资源可投入性匹配
- **输出**: 商机匹配报告（匹配度评分、推荐理由）

#### F-007 商机卡片生成
- **功能描述**: 生成标准化商机卡片，便于快速决策
- **卡片内容**:
  - 项目概要（名称、金额、业主、周期）
  - 商机分析（价值评估、竞争态势、匹配度）
  - 行动建议（负责团队、关键节点）
  - 相关信息链接（政策背景、招标文件）

### 1.4 信息推送模块

#### F-008 多渠道推送
- **功能描述**: 将商机信息推送给目标用户
- **推送渠道**:
  - 系统站内消息
  - 企业微信/钉钉
  - 邮件
- **推送规则**:
  - 实时推送（高优先级商机）
  - 定时汇总推送（每日/每周报告）
  - 支持用户自定义订阅规则

## 二、P1 重要功能

### 2.1 用户管理模块

#### F-009 用户账号体系
- **功能描述**: 支持多级账号管理
- **账号类型**:
  - 超级管理员（全系统配置）
  - 省级管理员（省内数据权限）
  - 普通用户（查看权限）
- **功能**: 账号创建、权限分配、使用统计

#### F-010 订阅配置
- **功能描述**: 用户自定义关注内容
- **配置维度**:
  - 关注地区（省/市）
  - 关注行业/项目类型
  - 关注金额范围
  - 关键词订阅
- **效果**: 个性化推送内容

### 2.2 数据可视化模块

#### F-011 数据看板
- **功能描述**: 可视化展示关键数据指标
- **看板内容**:
  - 专项债发行趋势图
  - 项目地区分布图
  - 商机漏斗图
  - 商机转化率统计
- **交互**: 支持下钻、筛选、导出

#### F-012 地图展示
- **功能描述**: 在地图上展示项目和商机分布
- **图层**:
  - 项目分布热力图
  - 商机状态标记
  - 区域竞争态势
- **交互**: 点击查看详情、区域筛选

### 2.3 作战SOP模块

#### F-013 SOP知识库
- **功能描述**: 积累各类项目的作战方法论
- **内容分类**:
  - 行业解决方案SOP（智慧城市、智慧医疗等）
  - 项目阶段SOP（立项跟踪、投标响应、实施交付）
  - 竞争应对SOP
- **来源**: 内部沉淀、外部采购、专家整理

#### F-014 SOP智能推荐
- **功能描述**: 根据商机卡片自动推荐相关SOP
- **推荐逻辑**:
  - 项目类型匹配
  - 地区经验匹配
  - 竞争态势匹配
- **效果**: 缩短方案准备时间

### 2.4 搜索与检索模块

#### F-015 全文检索
- **功能描述**: 支持对历史数据的全文搜索
- **搜索范围**: 项目、政策、招标、商机
- **搜索能力**: 
  - 关键词搜索
  - 组合条件筛选
  - 搜索结果高亮
- **高级功能**: 同义词扩展、智能推荐

### 2.5 竞争分析模块

#### F-016 竞争对手追踪
- **功能描述**: 追踪主要竞争对手的项目参与情况
- **追踪内容**:
  - 中标项目统计
  - 重点区域布局
  - 产品解决方案动向
- **输出**: 竞争态势报告

## 三、P2 增值功能

### 3.1 高级分析模块

#### F-017 商机价值预测
- **功能描述**: 预测商机的中标概率和价值
- **预测维度**:
  - 项目竞争激烈程度
  - 历史中标率分析
  - 客户关系评估
- **应用**: 优先级排序、资源分配决策

#### F-018 趋势预测分析
- **功能描述**: 基于历史数据预测行业趋势
- **预测内容**:
  - 专项债投向趋势
  - 区域投资热点
  - 技术演进方向

### 3.2 协同工作模块

#### F-019 商机协作空间
- **功能描述**: 为商机建立协作工作区
- **功能**:
  - 团队组建与任务分配
  - 文件共享与版本管理
  - 沟通记录与会议纪要
  - 进度跟踪与汇报
- **集成**: 与企业微信/钉钉集成

#### F-020 项目跟进提醒
- **功能描述**: 自动提醒关键节点
- **提醒内容**:
  - 招标截止日期
  - 开标时间
  - 合同签订期限
  - 项目里程碑
- **方式**: 系统消息、IM、日历同步

### 3.3 系统集成模块

#### F-021 CRM系统对接
- **功能描述**: 与运营商现有CRM系统对接
- **对接内容**:
  - 商机数据同步
  - 客户信息关联
  - 销售漏斗同步
- **技术**: API对接、数据映射

#### F-022 OA系统对接
- **功能描述**: 与OA系统集成
- **对接内容**:
  - 审批流程对接
  - 公文归档
  - 任务分派

### 3.4 报告与导出模块

#### F-023 自动报告生成
- **功能描述**: 自动生成周期性报告
- **报告类型**:
  - 日报（新增项目、关键动态）
  - 周报（商机进展、转化统计）
  - 月报（行业分析、竞争态势）
- **格式**: PDF、Word、PPT

#### F-024 数据导出
- **功能描述**: 支持数据批量导出
- **格式**: Excel、CSV、PDF
- **权限**: 按用户权限控制导出范围

### 3.5 移动端适配

#### F-025 移动端小程序
- **功能描述**: 微信小程序/企业微信应用
- **核心功能**:
  - 商机卡片浏览
  - 推送消息接收
  - 简单搜索查询
  - 审批处理
- **设计**: 轻量化、高频场景优先

## 四、功能依赖关系

```
数据采集(F-001~003) 
    → 数据处理(F-004~005) 
    → 商机分析(F-006~007) 
    → 信息推送(F-008)
    
用户管理(F-009) → 订阅配置(F-010) → 信息推送(F-008)

数据可视化(F-011~012) 依赖 数据处理(F-004~005)

作战SOP(F-013~014) 依赖 商机卡片(F-007)

高级分析(F-017~018) 依赖 历史数据积累

移动端(F-025) 依赖 核心功能完善
```

## 六、技术架构建议

### 数据层
- **数据采集**: Scrapy/Playwright + 代理池
- **数据存储**: PostgreSQL（结构化） + ES（全文检索） + MinIO（文件）
- **数据处理**: Python + NLP库（jieba, transformers）

### 服务层
- **后端**: Python FastAPI / Java Spring Boot
- **任务调度**: Celery / XXL-Job
- **消息队列**: RabbitMQ / Kafka

### 应用层
- **前端**: Vue3 / React
- **移动端**: 微信小程序原生 / Taro
- **推送**: 企业微信API / 钉钉API

### 基础设施
- **部署**: Docker + K8s
- **监控**: Prometheus + Grafana
- **日志**: ELK Stack
